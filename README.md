# HR Executive Analytics Dashboard

This project presents a comprehensive HR analytics solution built using Microsoft Power BI. It provides actionable insights to support data-driven decision-making in workforce planning, employee performance, compensation strategy, recruitment, and retention.

---

## Project Overview

The dashboard consolidates multiple HR data tables, enabling executives and HR managers to analyze trends, monitor KPIs, and identify improvement areas across the employee lifecycle.

**Primary Objective**
To deliver a centralized view of employee metrics and workforce trends through interactive and dynamic Power BI reports.

---

## Key Features

- Executive HR KPI Summary View  
- Workforce Demographics and Diversity Insights  
- Employee Performance and Rating Analysis  
- Compensation and Salary Benchmarking  
- Absenteeism and Leave Trends  
- Recruitment Funnel and Hiring Insights  
- Attrition, Retention, and Turnover Monitoring  

---

## Report Sections

| Report Page | Description |
|-------------|-------------|
| Executive Dashboard | Consolidated KPI metrics and YoY comparison |
| Employee Demographics | Age, gender, department, marital status distribution |
| Performance Report | Performance rating analysis and high performer ratio |
| Salary & Compensation | Salary growth, benchmarking, pay distribution |
| Absenteeism Report | Leave trends, absence behavior, department comparison |
| Recruitment Analytics | Hiring pipeline, selection ratio, cost trends |
| Attrition & Retention | Turnover trend, exit insights, risk visualization |

---

## Data Model

The data model follows a star schema including:

**Fact Tables**
- Employee Master
- Performance Reviews
- Salary History
- Leave Records

**Dimension Tables**
- Department
- Job Titles
- Calendar
- Location

## How to Use

1. Download the `.pbix` file from the repository.
2. Open the file in Microsoft Power BI Desktop (latest version).
3. Refresh the data model if needed.
4. Explore the interactive visuals and filters.
